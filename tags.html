"html" - корневой тег, который сообщает браузеру, что это HTML-документ. Все остальные элементы помещаются внутри него.

"head"— контейнер, в который помещаются метаданные документа, не видимые пользователям, но считываемые поисковыми роботами: например, "title" или "style".

"body" — тег, обрамляющий видимую пользователям часть документа. Всё, что вы укажите внутри этого контейнера, отобразится на странице.

"title"— метатег, который задаёт название страницы, отображаемое на вкладке браузера.

"header" — определяет содержимое блока с вводной информацией сайта или группой ссылок.

"h1"......"h6". — теги заголовков, от самого большого к самому маленькому.

"b" — жирный текст без придания важности выделенному фрагменту.

"strong" — расставление акцентов в тексте путём выделения его фрагментов полужирным начертанием.

"i" — выделение текста курсивом без придания важности.

"mark" — выделение частей текста жёлтым маркером.

"sub" — подстрочное начертание символов.

"sup" — надстрочное начертание символов.

"p" — контейнер для абзаца.

"br" — переносит текст на другую строку без создания абзаца.

"ol — список с цифрами.

"ul" — список со значками.

"li" — отметка каждого элемента перечня (цифра или значок в зависимости от типа списка).

"img" — вставка изображения. Имеет атрибут src, который указывает на адрес нужного файла.

"audio" — вставка звукового контента.

"video" — вставка видео (поддерживается Ogg, WebM и MP4).

"hr" — размещает на странице горизонтальную линию.

"form" — создание форм на странице. С помощью HTML описывается только внешний вид, для выполнения функций требуется запуск соответствующих скриптов на сервере.

"option" — описывает каждый отдельный пункт меню.

"input" — формирует поля для добавления пользовательских данных.

"output" — выводит результаты вычислений, сделанных с помощью скрипта.

"label" — оформляет пометку для поля, созданного тегом "input".

"textarea" — создаёт большие поля для ввода текста.
